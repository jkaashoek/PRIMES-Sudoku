About Us

This project was created as part of the MIT PRIMES program. To go to the PRIMES website, click here. The ultimate goal of this project is to help students learn programming. The website will be used to test our algorithm because we wish to use the same algorithm that creates the sudoku boards to help students learn how to program.
Part of the goal of the sudoku section of this project was to find a way to classify the difficulty of sudoku puzzles. This is why the homescreen does not have the simple easy, medium, or hard selection that you would normally see on a sudoku website. We want to be able to classify puzzles in their different difficulties based on a set of constrains (i.e. how many empty squares, density of given numbers, number of solutions etc.).
